Columnas de ancho automatico

Note: Try to add a new div with class="col" inside the row class - this will create four equal-width columns.

El container fluido siempre ocupa el ancho de la pagina y se adapta si esta se expande o se contrae.

.col1
.col2
.col3
.col4

Columnas de ancho definido

El container definido tiene un ancho maximo definido en la pagina y se adapta si esta se contrae.

.col1, 4col ancho
.col2, 3col ancho
.col3, 5col ancho

La suma de las tres columnas da un total de 12 columnas

Columnas de ancho definido

El container definido tiene un ancho maximo definido en la pagina y se adapta si esta se contrae.

.col1, 4col ancho, sm >= 576px
.col2, 8col ancho
